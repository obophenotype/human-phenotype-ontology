# Creating Phenopackets: 

There are several ways to create phenopackets. 

## PhenopacketLab

PhenopacketLab is an Angular/Springboot web application for the loading, editing, saving of data that follows the Phenopacket Schema.

- :material-account-box-outline: **PhenopacketLab**  
  <small>Web-based patient phenotyping & family history recording</small>  
  [:material-open-in-new: GitHub](https://github.com/TheJacksonLaboratory/PhenopacketLab){ target="_blank" }


## Phenoboard

Phenoboard is a tauri app that is available with installers for Mac, Windows, and Linux. It is intended to be used to curate cohorts of individuals and is the application used by the HPO project for biocuration.

- :material-account-box-outline: **Phenoboard**  
  <small>Curation of individuals with Human Phenotype Ontology and GA4GH Phenopacket Schema. </small>  
  [:material-open-in-new: GitHub](https://github.com/P2GX/phenoboard){ target="_blank" }
